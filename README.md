# Keto Macro Calculator

A small installable PWA for calculating keto macros on a phone. Height is entered in feet and inches, and weight is entered in stones plus pounds.

## Deploy on GitHub Pages

1. Upload these files to the `main` branch of `https://github.com/sourmilkman/ketokarma`.
2. In GitHub, open **Settings > Pages**.
3. Set **Source** to **Deploy from a branch**.
4. Choose `main` and `/root`, then save.
5. Visit `https://sourmilkman.github.io/ketokarma/`.

On Android Chrome, open the site and use **Install app** or **Add to Home screen**. The app caches itself after the first visit and works offline from then on.
